# 1.TCP:IP详解(卷1)
![TCP:IP详解(卷1)](./1.TCP:IP详解(卷1).jpg)

本书被若干严肃的书籍和论文引用。百度百科的网络相关的词条是一字不错的抄自本书。这可以说明本书原版和翻译版质量之高，本书通过网络运行的现象解释网络协议的工作原理，独辟蹊径，非常高明。 

第一卷必读。

注意，认准第一版。第二版的更新非常啰嗦，翻译质量也不敢恭维。
